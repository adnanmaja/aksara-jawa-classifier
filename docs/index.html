About aksara jawa scanner

Aksara Jawa Scanner is an AI-powered tool specifically built to recognize and analyze handwritten Javanese script (Aksara Jawa). Using deep learning technology with a ResNet18 architecture, the system can accurately identify traditional Javanese characters and provide their transliteration.

Currently in beta version. More features and improvements are coming soon! Check out my GitHub or social media below for feedback and contributions!
